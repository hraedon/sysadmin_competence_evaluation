This script does two things.

The first part queries Active Directory for disabled accounts that haven't logged in within a cutoff period, and exports them to a CSV file. The cutoff is hardcoded to 30 days in the script body.

The second part removes all AD group memberships from those same accounts. This is a write operation against Active Directory — the script doesn't just report, it modifies group membership.

The script touches:
- Active Directory (read queries, then write operations via Remove-ADGroupMember).
- The local filesystem (creates or overwrites a CSV file at $OutputPath).
