SVC-MONITORING — Critical. Domain Admin membership on a monitoring service account is a critical misconfiguration. Monitoring agents have broad network access by design — they poll all hosts, query WMI, and run with high uptime. A compromised monitoring platform or a credential spray against an 847-day-old password immediately yields Domain Admin. The AD note ("required for WMI access") is factually wrong. WMI access can be delegated at the namespace level without Domain Admin. This is either an uninvestigated assumption or a convenience shortcut that was never corrected. Either way, it's a stale, unnecessary Tier 0 exposure.

TIER0-ADMIN-RWALSH — Critical. The most recent interactive logon is on WS-RWALSH-01, a Tier 2 workstation. Tier 0 credentials cached in LSASS on a Tier 2 device can be harvested by any process running with local admin rights — browser exploit, phishing payload, or malicious macro. The shared Outlook profile on the workstation is a distinct and more serious exposure: this isn't just incidental workstation use, it's documented that the Tier 0 account is reading email on a general-purpose machine, which means it is exposed to every attachment and link that arrives in that mailbox. These are two separate vectors: LSASS harvest via local admin process, and email-delivered compromise of the Tier 0 account itself.

HELPDESK-LEAD — High. "Reset Password" delegated at the domain root propagates to the Domain Controllers OU. The ability to reset accounts in the DC OU — including the built-in Administrator account and the DSRM password — is effectively a Tier 0 capability. This delegation scope exceeds what a helpdesk lead role requires by a significant margin.

SVC-BACKUP — Correctly configured. Backup Operators is the least-privilege built-in group for backup software — necessary VSS and file access rights without Domain Admin. Interactive logon disabled. Logon rights scoped to backup servers only. Password rotated 23 days ago with enforced policy. This is the standard against which the other accounts should be measured.
