This is a very strong backup strategy. It follows the 3-2-1 rule by having an on-site NAS, off-site Azure Blob storage, and offline LTO-8 tapes. The use of immutable storage in Azure is a great security feature that prevents ransomware from deleting the backups. Using SSO and MFA for all IT staff is also a best practice that simplifies management while keeping things secure. This architecture meets all requirements for a secure off-site and immutable backup.
