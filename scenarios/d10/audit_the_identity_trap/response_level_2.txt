The backup plan is solid because it covers multiple locations and media types. The Azure Blob storage with immutability is the best part because it protects against data deletion. However, relying on a single region like East US for the off-site copy could be a risk if that whole Azure region has an outage. Also, using SSO for everything is good, but if the main corporate network is down, it might be hard for the admins to log in to the portal to start the recovery. I would suggest having some alternative way to log in if SSO isn't working.
